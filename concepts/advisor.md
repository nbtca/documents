# 指导老师（宋光慧）

宋光慧是 NBTCA 现任指导老师，浙大宁波理工学院[计算机学院](/concepts/college)教师，研究方向为图像内容分析与理解、深度学习、神经网络与大数据处理。

## 简介

- 在高校从事教学科研约二十年。
- 主讲数据库系统、数据结构、Web 程序设计（JSP）等课程。
- 长期从事基于 Java 的移动端与服务端开发，有丰富的工程实践经验。

## 荣誉

- 校优秀班导师（2019–2020、2021–2022 学年）
- 宁波市优秀班主任（2019–2020）

## 与 NBTCA

作为指导老师，宋光慧为协会的活动开展、对外协作与院方沟通提供指导与把关。社团的组织与运转见[组织与沿革](/about/organization)。
