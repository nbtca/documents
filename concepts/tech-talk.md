# 技术分享会

技术分享会是 NBTCA 社内不定期举办的技术主题分享——由成员就自己擅长或正在做的方向开讲，规模小、专业性强。例如一次关于可观测性实践（日志、指标、链路追踪）的分享。

与面向全校、讲基础的 [CA101](/concepts/ca101) / [CA102](/concepts/ca102) 讲座不同，技术分享会更偏社内、进阶，也更随性。
