## 确认需求

搭建计算机硬件系统，需要挑选自己需要的计算机硬件，分为零件选取、整机购入；
无论何种方式购入硬件，最重要的是确定计算机使用目的：是为了游玩对于图形性能要求很高的大型游戏？还是希望长期低功耗运行的小型服务器？不同的使用目的决定了计算机硬件的工作场景和性能要求，加上实际愿意付出的预算，构成了最终需要采购的硬件清单；

### 常见使用场景与参考配置

下面列出几种典型使用场景及对应的硬件配置思路，仅供参考，具体配置应根据当时市场行情和实际预算调整。

| 使用场景           | CPU                                  | 内存             | 显卡               | 存储            | 预算参考     |
| ------------------ | ------------------------------------ | ---------------- | ------------------ | --------------- | ------------ |
| 日常办公/上网/影音 | 入门级（如 Intel i3 / AMD R5）       | 8-16GB DDR5      | 集成显卡即可       | 512GB SSD       | 2000-3000 元 |
| 编程开发           | 中端（如 Intel i5 / AMD R5）         | 16-32GB DDR5     | 集成显卡或入门独显 | 512GB-1TB SSD   | 3000-5000 元 |
| 游戏（中高画质）   | 中高端（如 Intel i5/i7 / AMD R5/R7） | 16-32GB DDR5     | 中高端独显         | 1TB SSD         | 5000-8000 元 |
| 视频剪辑/3D 渲染   | 高端（如 Intel i7/i9 / AMD R7/R9）   | 32-64GB DDR5     | 高端独显           | 1TB+ SSD + HDD  | 8000 元以上  |
| 家用轻量服务器/NAS | 低功耗 CPU                           | 8-16GB ECC/非ECC | 集成显卡           | 大容量 HDD 为主 | 视需求灵活   |

:::tip 提示
以上价格为仅含主机（不含显示器、外设）的大致参考。硬件市场价格波动较大，购买前建议查看最新的性价比推荐和评测。
:::

### 如何评估硬件参数

选购硬件时需要关注以下关键参数：

#### CPU 评估要点

- 核心数和线程数：多核有利于多任务和编译，游戏更看重单核性能；
- 主频与睿频：基础频率和最大加速频率；
- 功耗（TDP）：影响散热方案的选择；
- 接口兼容性：确认 CPU 与主板插槽一致（如 LGA 1700、AM5 等）。

#### 内存评估要点

- 容量：当前建议至少 16GB，开发和创作建议 32GB；
- 频率和时序：频率越高、时序越低越好，但要确认主板和 CPU 支持；
- 通道数：双通道（两根内存条）比单通道性能提升明显。

#### 显卡评估要点

- 显存容量：游戏建议 8GB 以上，AI/渲染建议 12GB 以上；
- 性能天梯图：各大硬件媒体都有显卡天梯图可供参考；
- 功耗与尺寸：确认电源功率足够且机箱能装下。

#### 存储评估要点

- 接口和协议：NVMe（M.2 接口）速度远快于 SATA；
- 顺序读写和随机读写速度：关注实际使用体验中更重要的随机读写能力；
- 寿命（TBW）：写入总量指标，正常使用基本不必担心。

#### 电源评估要点

- 额定功率：计算所有硬件总功耗后留 30%-50% 余量；
- 转换效率认证：80 Plus 铜牌及以上为佳；
- 品牌与质量：电源直接关系所有硬件安全，不可贪图便宜。

## 硬件组成

大抵上，计算机硬件系统包括如下各部件：

- 中央处理器，用于处理数据；
- 内存，用于在处理器和各硬件间传输数据；
- 硬盘，用于存储数据；
- 电源，用于给各硬件供电；
- 独立显卡，用于处理中央处理器不擅长处理的图形数据，在不需要大规模图形计算性能的情况下可以考虑选择带有集成核心显卡的处理器进而不选购独立显卡；
- 主板，所有硬件通过主板连接；
- 散热器，用于为 CPU 等发热部件散热，分为风冷和水冷；
- 机箱，用于安装和保护各硬件，同时影响散热气流组织。

## 购入技巧

如果是购买一台组装完成的成品计算机，你需要着重考虑价格是否在合理区间、能否满足需要、能否售后完善；

至于获取最新的计算机硬件评测信息和价格状况，眼下第三方的测试结果多入牛毛，至于购买途径更是多不胜数，重点是应当多看多对比，即使经验丰富的采购人员也会翻车。在预算紧张时，也可以考虑观察是否购入二手硬件，相应的，也需要更加详细的评估硬件以免购入不如意的硬件。

### 选购渠道

- **线上自营平台**：京东自营、天猫旗舰店等，正品有保障，售后方便；
- **线下实体店**：可以实际看到和摸到硬件，但要警惕加价和偷换配件的情况；
- **二手平台**：闲鱼、转转等，价格有优势但需要有一定鉴别能力；
- **海外购买**：部分硬件在海外渠道价格更优，但需考虑保修和关税。

### 二手硬件评估

:::warning 注意
购买二手硬件时务必当面或通过视频验货，确认硬件功能正常。
:::

- **CPU**：故障率极低，二手性价比高，注意检查针脚/触点是否完好；
- **内存**：注意检查颗粒品牌和频率是否与标注一致，运行 MemTest 验证稳定性；
- **显卡**：风险最大的二手硬件，矿卡泛滥，注意检查显卡外观磨损、风扇噪音、满载温度；
- **硬盘**：使用 CrystalDiskInfo 检查通电时间和健康状态，避免购入高使用时长的盘；
- **电源**：不建议购买二手电源，电源老化可能损坏其他硬件。

## 安装

组织计算机各部件工作，基本上遵循各个时代对于计算机硬件组织方式的约束，不同接口和协议的约束，整台机器供电和散热空间的考量，这些都会最终影响计算机硬件的组装，应当多参考同时期硬件安装人员的经验和各计算机硬件的安装手册。

### 安装顺序建议

推荐的台式机组装顺序如下：

1. **安装 CPU 到主板**：对齐防呆缺口，轻放入插槽，扣好压杆（注意不要触碰针脚/触点）；
2. **安装内存条**：对齐缺口方向，均匀用力按下至两侧卡扣自动锁定，双通道请隔槽插入；
3. **安装 M.2 SSD**：倾斜插入 M.2 插槽，按下固定螺丝；
4. **安装 CPU 散热器**：涂抹适量硅脂（约米粒大小），安装散热器并固定（对角线顺序拧紧螺丝）；
5. **将主板安装到机箱**：先装好挡板，对齐铜柱孔位，拧紧固定螺丝；
6. **安装电源**：根据机箱结构安装到对应位置；
7. **安装显卡**：插入 PCIe 插槽，拧紧挡板螺丝，接上电源线；
8. **连接线缆**：主板供电（24pin + 8pin CPU）、显卡供电、SATA 数据线和电源线、前面板线（开机键、重启键、指示灯、USB、音频）；
9. **理线**：整理线缆避免遮挡风道，确保散热畅通。

### 常见错误与注意事项

:::warning 装机常见错误
以下是新手装机最容易犯的错误，务必注意避免：
:::

- **忘记装 I/O 挡板**：主板装入机箱后再想装挡板就需要拆下主板，非常麻烦；
- **内存没有插到位**：内存必须完全插入直到卡扣自动锁定，否则无法开机；
- **CPU 散热器没有撕掉底部保护膜**：会导致散热极差，CPU 迅速过热；
- **前面板跳线接错**：开机键和重启键不分正负，但 LED 指示灯分正负极；
- **忘记接 CPU 辅助供电**：只接 24pin 主供电而忘记接 CPU 8pin 供电线会导致无法开机；
- **显卡没有完全插入 PCIe 插槽**：需要确认插槽末端的卡扣已经锁定；
- **硅脂涂抹过多**：过多的硅脂会溢出到主板上，可能导致短路，米粒大小即可；
- **螺丝拧过紧**：尤其是主板和散热器的螺丝，过紧可能导致 PCB 变形。

## 安装后测试与验证

硬件安装完成后，不要急于安装操作系统，先进行基础验证。

### 首次开机检查

1. **接通电源，按下开机键**：观察风扇是否正常转动（CPU 风扇、机箱风扇、显卡风扇）；
2. **观察主板指示灯/蜂鸣器**：如果不能正常进入 BIOS，主板上的 Debug 灯或蜂鸣器会提示故障所在部件（CPU/内存/显卡/存储）；
3. **进入 BIOS 界面**：确认 CPU 型号、内存容量和频率、存储设备是否被正确识别；
4. **开启 XMP/EXPO**：在 BIOS 中开启内存的 XMP（Intel）或 EXPO（AMD）以使内存运行在标称频率。

### 稳定性测试

安装操作系统后，建议运行以下测试确认硬件稳定性：

- **CPU 稳定性**：使用 AIDA64 或 Prime95 进行烤机测试，持续 15-30 分钟观察温度和是否报错；
- **内存稳定性**：使用 MemTest86 或 Windows 内存诊断工具检测内存是否有错误；
- **显卡稳定性**：使用 FurMark 或 3DMark 进行 GPU 压力测试；
- **硬盘性能**：使用 CrystalDiskMark 测试读写速度是否达到标称值；
- **综合测试**：运行一款大型游戏或渲染任务 1-2 小时，观察是否出现蓝屏、死机或异常。

:::tip 温度参考

- CPU 满载温度建议不超过 90°C，日常使用建议在 60°C 以下；
- GPU 满载温度通常在 70-85°C 范围内属于正常；
- 如果温度异常偏高，检查散热器安装是否到位、硅脂是否涂抹正确、机箱风道是否合理。
  :::

### 常见开机故障排查

| 现象                   | 可能原因                               | 排查方法                                         |
| ---------------------- | -------------------------------------- | ------------------------------------------------ |
| 按开机键无任何反应     | 电源未通电、前面板跳线未接、电源损坏   | 检查电源开关和线缆，尝试短接主板开机针脚         |
| 风扇转但无画面         | 内存未插好、显卡未插好、显示器线接错   | 重新插拔内存和显卡，确认显示器接在显卡而非主板上 |
| 反复重启               | CPU 供电未接、散热器未装好导致过热保护 | 检查 CPU 8pin 供电和散热器安装                   |
| 进入 BIOS 但找不到硬盘 | SATA/M.2 未接好、BIOS 中未启用对应接口 | 重新插拔存储设备，检查 BIOS 设置                 |
| 蓝屏或频繁死机         | 内存不兼容或损坏、驱动问题             | 运行内存测试，尝试单根内存逐一排查               |

## 日常维护建议

- **定期清灰**：建议每 6-12 个月清理一次机箱内部灰尘，重点关注散热器和风扇；
- **更换硅脂**：建议每 1-2 年更换一次 CPU 硅脂，保持良好散热；
- **关注温度**：使用 HWMonitor、AIDA64 等软件定期关注硬件温度；
- **固件更新**：SSD 固件和主板 BIOS 有重要更新时建议升级，但不要频繁升级；
- **电源保护**：使用带有浪涌保护的插座，雷雨天气建议断开电源。
