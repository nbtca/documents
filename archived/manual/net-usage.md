## 学习互联网使用的意义

如果了解互联网的历史就会发现，现今的互联网世界同早期构建互联网世界的梦想有些偏差，受到现实世界中地缘政治的影响、以及存在各地网络基础设施建设实践中的差异，各地区充分自由的交换信息绝非易事；

根据冯·诺伊曼的博弈论，我们可以了解到对于任何决策的考量很大程度上受到对于决策情况的了解程度，实际上这一点又同你所获得的信息来源的丰富程度强相关，那么我们可以得到一些浅显的结论：那就是，尽可能掌握更多更广泛的信息一定程度上对于我们的决策是有利的；

那么在信息时代，掌握国际互联网的使用就变成了一项人类必要的生存活动，因为当今的时代，我们很难说每个人完全不同互联网打交道。

## 如何使用互联网

### 浏览器的选择与使用

浏览器是访问互联网的主要工具，选择一个合适的浏览器是高效使用互联网的第一步。

#### 推荐的浏览器

- **Google Chrome**：市场份额最高，扩展生态丰富，开发者工具强大，但内存占用较高；
- **Mozilla Firefox**：开源、注重隐私保护，自定义程度高；
- **Microsoft Edge**：Windows 自带，基于 Chromium 内核，与 Windows 生态集成好；
- 建议避免使用国产"套壳"浏览器（如 360 浏览器、QQ 浏览器等），它们通常捆绑广告和推送。

#### 浏览器基本技巧

- `Ctrl + T`：新建标签页；`Ctrl + W`：关闭当前标签页；`Ctrl + Shift + T`：恢复刚关闭的标签页；
- `Ctrl + L`：聚焦地址栏，可直接输入网址或搜索内容；
- `Ctrl + F`：页面内搜索关键词；
- `F12`：打开开发者工具，可用于查看网页源代码和网络请求；
- 使用书签功能收藏常用网站，通过书签栏快速访问。

#### 浏览器安全设置

- 保持浏览器更新到最新版本；
- 不要随意安装来源不明的浏览器扩展；
- 推荐安装的扩展：uBlock Origin（广告拦截）；
- 定期清理浏览器缓存和 Cookie。

:::warning 注意
不要在公共电脑上保存密码和登录状态，使用完毕后清除浏览数据或使用隐私模式浏览。
:::

### 搜索引擎的高效使用

搜索引擎是获取信息的最重要工具，掌握搜索技巧可以大幅提升信息获取效率。

#### 选择搜索引擎

- **Google**：全球最大的搜索引擎，索引最全面，英文搜索结果质量最高；
- **Bing**：微软的搜索引擎，国内可直接访问，搜索质量较好；
- **百度**：中文搜索市场占有率最高，但搜索结果中广告较多，需注意甄别。

#### 搜索技巧

- **使用双引号精确搜索**：`"蓝屏代码 0x0000007E"` 会精确匹配这个完整短语；
- **使用减号排除关键词**：`Python 教程 -广告 -培训` 排除包含"广告"和"培训"的结果；
- **指定站内搜索**：`site:github.com NBTCA` 只在 GitHub 中搜索 NBTCA 相关内容；
- **指定文件类型**：`filetype:pdf 操作系统原理` 只搜索 PDF 格式的文档；
- **使用英文搜索**：技术问题建议优先使用英文关键词在 Google 上搜索，通常能找到更专业的解答。

:::tip 提示
遇到技术问题时，直接搜索错误信息（尤其是英文错误代码）往往比描述问题现象更有效。Stack Overflow、GitHub Issues 和各类技术论坛是高质量技术答案的常见来源。
:::

## 网络故障排查基础

在日常维修中，网络问题是用户最常遇到的问题之一。

### 基本排查思路

遇到网络问题时，按以下顺序逐步排查：

1. **检查物理连接**：网线是否插好？WiFi 是否已打开？飞行模式是否关闭？
2. **检查网络适配器**：设备管理器中网卡是否正常（有无黄色感叹号）？
3. **检查 IP 配置**：运行 `ipconfig`，确认是否获取到正确的 IP 地址；
4. **测试本地连通性**：`ping 127.0.0.1`（测试本机网络栈）→ `ping 网关地址`（测试局域网）→ `ping 8.8.8.8`（测试外网）→ `ping baidu.com`（测试 DNS）；
5. **尝试重置网络**：`netsh winsock reset` 和 `ipconfig /flushdns`，然后重启。

### 常见网络问题

| 现象                 | 可能原因                     | 解决方法                                |
| -------------------- | ---------------------------- | --------------------------------------- |
| 完全无法联网         | 网卡驱动缺失、硬件故障       | 检查设备管理器，重新安装网卡驱动        |
| 能连 WiFi 但无法上网 | DNS 配置错误、IP 冲突        | 手动设置 DNS，释放并重新获取 IP         |
| 网页打不开但 QQ 能用 | DNS 解析失败                 | 更换 DNS 服务器（223.5.5.5 或 8.8.8.8） |
| 网速异常缓慢         | 带宽被占用、信号差           | 检查是否有后台下载，尝试靠近路由器      |
| 校园网无法认证       | 认证客户端问题、MAC 地址绑定 | 重新安装认证客户端，检查设备注册        |

### 校园网使用提示

- 校园网通常需要通过 Web Portal 或专用客户端认证后才能上网；
- 注意校园网账号的设备数量限制，超出限制时需要先下线其他设备；
- 校园网通常对 P2P 下载有限制，使用时注意合规。

## VPN 与代理

### 基本概念

- **VPN（虚拟专用网络）**：在公共网络上建立加密隧道，使通信数据受到保护。企业常用 VPN 让远程员工安全访问公司内网；
- **代理服务器**：作为客户端和目标服务器之间的中转，代理服务器代替客户端发送请求并转发响应；
- **正向代理**：客户端通过代理访问外部资源（如访问国际学术资源）；
- **反向代理**：服务器端使用代理接收和分发请求（如 Nginx 反向代理）。

### 学术资源访问

- 许多国际学术数据库和资源需要通过学校图书馆的 VPN 或代理访问；
- 学校通常提供 WebVPN 或 VPN 客户端，用于校外访问校内资源；
- 善用 Google Scholar、Sci-Hub、学校图书馆数据库等渠道获取学术文献。

## 网络安全与隐私保护

在互联网上保护个人信息安全是每个用户的必修课。

### 密码安全

- 使用强密码：至少 12 位，包含大小写字母、数字和特殊字符；
- 不同网站使用不同密码，避免"一个密码走天下"；
- 使用密码管理器（如 Bitwarden、KeePass）管理密码；
- 启用两步验证（2FA）：在支持的网站上启用，大幅提高账号安全性。

### 防范常见威胁

- **钓鱼攻击**：仔细检查邮件发件人地址和链接网址，不要点击可疑链接，不要在陌生网站输入个人信息；
- **恶意软件**：不下载来路不明的软件，保持系统和杀毒软件更新；
- **社会工程**：不要轻信陌生人索要个人信息或账号密码的请求，即使对方自称是官方客服；
- **公共 WiFi 安全**：在公共 WiFi 环境下避免进行敏感操作（如网银、支付），必要时使用 VPN。

:::warning 警惕
电信诈骗形式多样，牢记：不轻信、不转账、不泄露个人信息。遇到疑似诈骗及时向学校保卫处和公安机关反映。
:::

### 个人隐私保护

- 注意各类应用的权限申请，不要授予不必要的权限（如天气应用请求通讯录权限）；
- 定期检查并清理社交媒体上的个人信息暴露；
- 使用浏览器隐私模式浏览不想留下记录的内容；
- 谨慎使用公共设备，使用后及时退出登录并清除浏览数据；
- 了解并合理使用各平台的隐私设置。
