# C盘清理标准化流程

[[toc]]

## 概述

本文档提供 C 盘空间不足时的标准化清理流程，帮助用户安全、高效地释放磁盘空间。

## 检查清单

在开始清理前，请确认以下事项：

- [ ] 已备份重要数据
- [ ] 已关闭正在运行的程序
- [ ] 已确认当前 C 盘可用空间
- [ ] 已了解用户对空间释放的预期

## 基础清理流程

### 1. 使用系统自带工具

#### 磁盘清理

1. 右键点击 C 盘 → 属性 → 磁盘清理
2. 点击「清理系统文件」获取更多选项
3. 勾选需要清理的项目（如 Windows 更新清理、临时文件等）
4. 点击确定执行清理

#### 存储感知

1. 打开「设置」→「系统」→「存储」
2. 开启「存储感知」自动清理临时文件
3. 点击「临时文件」手动清理

### 2. 使用 Dism++ 清理

1. 以管理员身份运行 Dism++
2. 选择「空间回收」功能
3. 勾选需要清理的项目
4. 点击「清理」执行

### 3. 使用 WizTree 分析

1. 运行 WizTree，选择 C 盘进行扫描
2. 查看占用空间较大的文件夹
3. 定位不需要的大文件进行清理

## 常见清理项目

### 用户数据迁移

- 桌面文件夹、下载文件夹、文档文件夹迁移至 D 盘：使用文件资源管理器右键属性 → 位置 → 移动
- 游戏等已安装程序的目录迁移：使用符号链接（mklink /J）或[junction](https://docs.microsoft.com/en-us/sysinternals/downloads/junction)

### 软件缓存清理

- 浏览器缓存
- 微信/QQ 缓存文件
- 软件临时文件

### 系统文件清理

- Windows 更新缓存
- 系统还原点（谨慎操作）
- 休眠文件（如不需要休眠功能）

## 高级操作

### 把 D 盘压缩后腾出空间给 C 盘

:::warning 注意事项
此操作涉及分区调整，请务必提前备份重要数据！
:::

#### 方法一：系统自带工具

1. 按 `Win + X`，选择「磁盘管理」
2. 右键 D 盘 → 压缩卷
3. 输入要压缩的空间大小
4. 将未分配空间分配给 C 盘（需要相邻）

#### 方法二：DiskGenius

1. 下载并运行 DiskGenius
2. 选择 D 盘 → 调整分区大小
3. 将多余空间分配给 C 盘
4. 应用更改并重启

## 注意事项

- 清理前务必备份重要数据
- 不要随意删除系统文件夹中的文件
- 分区操作建议在 PE 环境下进行
- 如不确定某个文件是否可删除，请先咨询或搜索
