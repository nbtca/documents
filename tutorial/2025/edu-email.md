## 详细教程

### 阿里云

- **用途**
阿里云大学生优惠活动，每年可以获得300元代金券，用于购买指定云服务器等服务。
- **步骤**
访问<https://university.aliyun.com>
<https://university.aliyun.com/buycenter/>
完成教育认证，无门槛。

- **实测**
每年300代金券，余额形式，花多少扣多少，可以用来抵扣阿里云百练大模型API调用，可购买一年2C0.5G海外服务器，国内服务器一年似乎要加几十元。

### JetBrains 全家桶

- **用途**

JetBrains 提供免费的全家桶许可证给符合条件的学生和教师，包含 IntelliJ IDEA、PyCharm、WebStorm 等多款开发工具。

- **步骤**

1. 访问 [JetBrains 学生页面](https://www.jetbrains.com/student/)
2. 使用教育邮箱注册账号
3. 按照指引完成验证，即可免费使用 JetBrains 全家桶

- **实测**

2025.12.12：目前@nbt.edu.cn 结尾的邮箱可以直接通过验证。

### GitHub Student Developer Pack（GitHub Copilot）

- **用途**
GitHub Student Developer Pack 提供众多开发工具和服务的免费或优惠使用权限，包括 GitHub Copilot、DigitalOcean、Namecheap 等。

> 可以获得至少两个域名（一年）、GitHub Copilot 每月300额度的Premium Request使用额度。
> 其中DigitalOcean需要PayPal或者银行卡充值$5激活。激活后有$200额度。

- **步骤**

0. GitHub个人设置中添加@nbt.edu.cn 结尾的教育邮箱，以便后续验证。在GitHub Profile Settings -> Emails 中添加并验证邮箱。修改个人信息比如介绍，可以写些内容使得看起来像真人。
1. 访问 [GitHub Education](https://education.github.com/pack)
2. 使用教育邮箱登录或注册 GitHub 账号
3. 点击「Get your Pack」按钮，按照指引提交申请

- **实测**

2025：无法直接通过，尝试以下方法可以通过

1. 学信网点击生成学籍证明（中文）
2. 使用Edge/Chrome打开生成的PDF，使用浏览器翻译功能，将页面翻译成英文
3. 在机房电脑课、宿舍等校内网络环境，不挂梯子的情况下，使用手机摄像头拍摄电脑屏幕展示的「学籍证明」，通常可以通过验证。

> - 提示
> 直接上传截图可能会提示重复。
> GitHub大概率是机器审核，如果没成功大概率5分钟就能看到结果，尝试不同方式重新提交申请即可。

### Azure for Students（微软云服务）

- **用途**
Azure for Students 提供免费的云计算资源。主要是海外云服务器。每年有$100的免费额度，激活一年在最后一个月会提示续期，续期后余额会恢复$100。

- **步骤**

1. 访问 [Azure for Students](https://azure.microsoft.com/en-us/free/students/)
2. 使用教育邮箱注册账号
3. 按照指引完成验证

- **实测**

建议使用`@nit.zju.edu.cn`可以通过，如果第一次完成各种信息的填写后显示没有资格，不要慌，可以过几天或一周后来看，再登录可能就激活了。
